# FERMO_10.25
Desarrollo de aplicaciones para Fermo en Odoo v17 CE
PROYECTO ODOO V17 CE – EMPRESA FINANCIERA
He desarrollado un proyecto para la administración de una financiera.
El proyecto utiliza como base Odoo v17 CE y no utiliza la mayoría de los módulos nativos de Odoo (a excepción de contactos, ajustes y otros). No utiliza: compras, venta, inventario, contabilidad, etc; sino que utiliza un desarrollo propio montado sobre la base de Odoo).
Consiste en el desarrollo de 5 módulos que interactúan entre sí:
1.	custom_contact_types
2.	chequera
3.	divisas
4.	sucursales_cajas
5.	commission_management
